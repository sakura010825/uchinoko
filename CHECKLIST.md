# 動作確認チェックリスト

プロジェクトをデプロイする前に、以下の項目を確認してください。

## 環境設定

- [ ] `.env.local`ファイルが作成されている
- [ ] Firebase設定値が正しく設定されている
- [ ] Gemini APIキーが設定されている
- [ ] 環境変数名が`NEXT_PUBLIC_`で始まっている（クライアント側で使用する場合）

## Firebase設定

- [ ] Firebaseプロジェクトが作成されている
- [ ] Firestoreデータベースが作成されている
- [ ] Storageバケットが作成されている
- [ ] Authenticationが有効になっている（Email/Password）
- [ ] Firestoreセキュリティルールがデプロイされている
- [ ] Storageセキュリティルールがデプロイされている
- [ ] Firestoreインデックスが作成されている
