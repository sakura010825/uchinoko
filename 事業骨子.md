# プロジェクト概要
* **アプリ名:** うちの子の気持ち（仮）
* **コンセプト:** 愛猫の写真を投稿すると、猫の性格（うちの子手帳）に基づいてAIが気持ちを代弁してくれる「翻訳SNS」。
* **デザイン:** 猫好きな女性が好む、ポップでかわいらしく、親しみやすい世界観。（特定の色に偏りすぎず、どんな毛色の猫ちゃんも映えるような優しいトーン）

# 技術スタック（個人開発向け・ダイエット構成）
* **Frontend:** Next.js 14 (App Router), TypeScript
* **UI Framework:** Tailwind CSS, Lucide React (アイコン), shadcn/ui (コンポーネント)
* **Backend / DB:** Firebase (Authentication, Firestore, Storage)
* **AI Integration:** Google Gemini API (画像解析とテキスト生成に使用)
* **Image Processing:** sharp (SNSシェア画像生成用)

# ユニークな機能・用語定義（重要）
以下の用語と仕様をコード内の変数名やUIの文言に反映してください。

1. **うちの子手帳 (Uchi-no-ko Techo):**
   - ユーザーが登録する猫のプロフィール情報（名前、性格、口調など）。
   - 「取扱説明書」ではなく、愛情のこもった「手帳」というコンセプト。

2. **肉球翻訳 (Nikukyu Translation):**
   - AI生成ボタンのこと。デザインは「ぷにぷにした肉球の形」にする。
   - **重要：ローディングアニメーション**
     - 一般的なスピナーではなく、「画面上に肉球の足跡がポン・ポン・ポン…とスタンプのように順番に押されていくアニメーション」を実装してください。

3. **カリカリ (Karikari):**
   - 「いいね」機能のこと。アイコンは魚やキャットフードの形。
   - 通知文言は「〇〇ちゃんがカリカリをもらいました！」とする。

4. **LINEスタンプチャレンジ:**
   - 投稿が8つ貯まると、背景透過PNGのスタンプセットを一括ダウンロードできる機能。
   - 「あと3個でスタンプが作れるよ！」といった進捗バーを表示してモチベーションを上げる。

5. **シェア機能 (Omikuji Share):**
   - SNSシェア時に、写真＋一言に加え「今日のラッキーアイテム」などがランダムで付与される画像合成機能を想定。
